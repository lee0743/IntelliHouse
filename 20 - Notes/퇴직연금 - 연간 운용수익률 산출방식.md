# 퇴직연금 - 연간 운용수익률 산출방식

> 주제 허브: [[퇴직연금]]
> 관련 노트: [[퇴직연금 - 자산운용 업무흐름]] · [[퇴직연금 - 운용지시 제약사항]]

---

## 1. 수익률 산출 목적

- **공시 의무**: 퇴직연금감독규정 제23조제3항 및 시행세칙 제7조에 따라 퇴직연금사업자는 매 분기 적립금 운용수익률을 공시
- **비교 기능**: 사업자 간 운용성과 비교 → 가입자·사용자의 사업자 선택 정보 제공
- **업무흐름 연결**: 7단계 자산운용 흐름 중 **⑥ 기준가 평가**가 수익률 산출의 직접 입력값

---

## 2. 수익률 공식

### ① 감독규정 공시 공식 (Korean Regulatory Standard)

퇴직연금감독규정 제23조제3항 · 시행세칙 제7조의 공식:

```
              직전 1년간 운용수익
수익률(%) = ─────────────────────────────── × 100
            직전 1년간 평균 적립금 (운용수익 제외)
```

- **수수료**: 적립금 총액 산정 시 제외
- **중장기**(3·5·7·10년): 연도별 수익률의 기하평균으로 산출

### ② Modified Dietz Method (개별 포트폴리오 성과 측정 / CFA Institute 표준)

현금흐름의 발생 시점을 반영하는 정밀 공식. 감독규정 공시 공식은 이 방법의 단순화 버전:

```
        기말 적립금 - 기초 적립금 - 순유입액
R(%) = ──────────────────────────────────────── × 100
        기초 적립금 + 기간가중 순유입액
```

| 항목 | 설명 |
|------|------|
| **순유입액** | 사용자부담금 + 근로자추가부담금 − 중도인출 − 연금수령액 |
| **기간가중 순유입액** | Σ [ 현금흐름_i × (잔여일수_i / 전체기간일수) ] |
| **잔여일수_i** | 현금흐름 발생일로부터 기말까지 남은 일수 |

> 기간 초에 유입된 자금은 가중치 높음 (오래 운용됨), 기간 말에 유입된 자금은 가중치 낮음.

## 3. 연환산 수익률

기간이 1년과 다를 경우 연 단위로 환산:

```
연환산 수익률 = (1 + 기간수익률)^(365/기간일수) − 1
```

예) 6개월(182일) 기간수익률 3% → 연환산: `(1.03)^(365/182) − 1 ≈ 6.09%`

---

## 4. DB형 vs DC형: 수익률의 의미 차이

| 구분 | DB형 (확정급여형) | DC형 / IRP |
|------|------------------|------------|
| **운용 주체** | 사용자(회사) | 가입자 본인 |
| **수익률의 의미** | 퇴직급여 부채 대비 적립금의 운용성과 | 가입자 개인 계좌의 실현 수익률 |
| **부족 적립 시** | 회사가 추가 납부 의무 | 가입자 손실로 귀결 |
| **ALM 관련성** | 자산-부채 듀레이션 매칭 필요 | ALM 불필요 |

→ 동일한 5% 수익률이라도 DB형은 부채 증가율과의 비교가 핵심이고, DC형은 가입자 노후 자산 증가가 핵심.

---

## 5. 상품 유형별 수익률 계산

| 상품 유형 | 수익률 산출 방식 |
|-----------|----------------|
| **원리금보장형** (정기예금, GIC 등) | 약정금리 기준 (확정 수익률) |
| **실적배당형** (펀드, ETF) | 기준가 변동 기반 (매 영업일 평가) |
| **혼합 포트폴리오** | 상품별 수익률의 편입 비중 가중평균 |

---

## 6. 산출 시 유의사항

- **총수익률 vs 순수익률**: 수수료·세금 차감 전후를 구분하여 비교
- **납입 시점 왜곡**: 기여금이 기간 말에 집중되면 단순 평균법에서 수익률이 과대/과소 산출될 수 있음 → Modified Dietz로 보정
- **벤치마크 비교**: 공시 수익률은 사업자별 BM 대비 초과수익률로도 평가

## My Thoughts

- 수익률 공식 구현 시 시스템 레벨에서 현금흐름 이벤트(납입·인출)의 타임스탬프를 정밀하게 기록해야 Modified Dietz 계산이 가능
- 감독규정 공시는 단순화된 공식이지만, 내부 성과 평가·ALM·리스크 관리에는 Modified Dietz 또는 TWR(Time-Weighted Return)을 사용하는 것이 표준
- ⑥ 기준가 평가 단계가 매 영업일 정확히 이루어져야 수익률 산출이 가능 → 기준가 오류는 수익률 오류로 직결
