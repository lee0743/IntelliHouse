### Q11. Modified Dietz 공식 빈칸 채우기

아래 Modified Dietz 수익률 공식의 빈칸을 채우시오.

```
        (         ) - 기초 적립금 - 순유입액
R(%) = ──────────────────────────────────────── × 100
        기초 적립금 + (              )
```

**기간가중 순유입액** = Σ [ 현금흐름 × ( \_\_\_\_ / \_\_\_\_ ) ]

<details>
<summary>정답 보기</summary>

```
        기말 적립금 - 기초 적립금 - 순유입액
R(%) = ──────────────────────────────────────── × 100
        기초 적립금 + 기간가중 순유입액
```

**기간가중 순유입액** = Σ [ 현금흐름 × ( **잔여일수** / **전체기간일수** ) ]

- 잔여일수: 현금흐름 발생일로부터 기말까지 남은 일수
- 기간 초에 유입된 자금일수록 가중치가 높음 (오래 운용됨)

</details>

### Q12. 순유입액 구성요소

순유입액의 정의를 완성하시오.

```
순유입액 = 사용자부담금 + (          ) - 중도인출 - (          )
```

<details>
<summary>정답 보기</summary>

```
순유입액 = 사용자부담금 + 근로자추가부담금 - 중도인출 - 연금수령액
```

- 유입: 사용자부담금, 근로자추가부담금 (적립금을 늘리는 항목)
- 유출: 중도인출, 연금수령액 (적립금을 줄이는 항목)

</details>

### Q13. 연환산 수익률 계산

6개월(182일) 기간수익률이 3%일 때, 연환산 수익률 공식과 계산 결과를 쓰시오.

<details>
<summary>정답 보기</summary>

**공식**: `연환산 수익률 = (1 + 기간수익률)^(365/기간일수) − 1`

**계산**: `(1.03)^(365/182) − 1 ≈ 0.0609 = 약 6.09%`

- 단순히 3% × 2 = 6%로 계산하면 복리 효과를 무시한 근사치
- 정확한 연환산은 기하급수(복리) 방식을 사용

</details>
