This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Important

The `old/` directory contains archived files from a previous version of this project. **Do not modify, reference, or treat those files as current.** They exist for historical reference only.

## Vault Structure

This repository is an Obsidian knowledge base (학습/지식베이스). The folder structure follows a Johnny Decimal + Zettelkasten hybrid:

| Folder | Purpose |
|--------|---------|
| `00 - Home/` | Dashboard and top-level MOC entry point |
| `10 - Inbox/` | Quick capture — unprocessed fleeting notes |
| `20 - Notes/` | Permanent notes (one concept per note) |
| `30 - Topics/` | Map of Content files, one per subject area |
| `40 - Resources/` | Source notes (books, articles, lectures) |
| `50 - Templates/` | Note and MOC templates for Obsidian |
| `.obsidian/` | Obsidian configuration — do not edit manually |

New notes default to `10 - Inbox/`. Processed notes move to `20 - Notes/` or `30 - Topics/`.
