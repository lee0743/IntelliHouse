# CLAUDE.md — JSP 볼트

This file provides guidance to Claude Code when working inside the `JSP/` folder.
This folder is a standalone Obsidian vault for studying JSP (JavaServer Pages).

## Vault Structure

| Folder | Purpose |
|--------|---------|
| `Notes/` | Permanent notes (one concept per note) |
| `Topics/` | Map of Content (MOC) file — `JSP.md` |
| `Templates/` | Note and MOC templates for Obsidian |
| `.obsidian/` | Obsidian configuration — do not edit manually |

New notes go directly to `Notes/`. The MOC is at `Topics/JSP.md`.
