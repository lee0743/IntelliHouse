## Important

The `old/` directory contains archived files from a previous version of this project. **Do not modify, reference, or treat those files as current.** They exist for historical reference only.

## Vault Structure

This repository is an Obsidian knowledge base (학습/지식베이스). Each subject has its own top-level folder containing `Notes/` and `Topics/` subfolders.

| Folder | Purpose |
|--------|---------|
| `00 - Home/` | Dashboard and top-level MOC entry point |
| `10 - Inbox/` | Quick capture — unprocessed fleeting notes |
| `40 - Resources/` | Source notes (books, articles, lectures) |
| `50 - Templates/` | Note and MOC templates for Obsidian |
| `<주제>/Notes/` | Permanent notes for that subject (one concept per note) |
| `<주제>/Topics/` | Map of Content (MOC) file for that subject |
| `.obsidian/` | Obsidian configuration — do not edit manually |

Current subjects: `JSP/`, `퇴직연금/`

New notes default to `10 - Inbox/`. Processed notes move to `<주제>/Notes/` and the subject MOC lives in `<주제>/Topics/`.
